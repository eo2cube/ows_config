Multi year-averages and the respective standard deviations of vegetation indices for 16 crop classes in 5 day-intervals from March 3rd to October 28th across Mecklenburg-Vorpommern. 

Each CSV file contains data for one vegetation index (specified in file name) with the averages (means) and the standard deviations (stds) being stored in seperate files).

File naming scheme:		*VegetationIndex*_*MeansOrStds*_*FirstYear*-*LastYear*.csv


Author: Christoph Schierghofer (DLR Oberpfaffenhofen)
